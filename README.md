# React -Tehtävät ja harjotustyö

**Setup terminaalista:**

- npm install
- npm install axios
- npm install json-server

- npm install -g react-devtools

---

**Json serveri päälle:**
- npx json-server --port=3000 --watch db.json

**Run:**
- npm run dev

---

Tehtävät:
https://fullstackopen.com/osa2/
